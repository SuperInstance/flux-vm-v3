# Contributing to flux-vm-v3

## Getting Started

```bash
git clone https://github.com/SuperInstance/flux-vm-v3.git
cd flux-vm-v3
cargo test
```

## Workflow

1. Fork or branch from `main`
2. Make changes
3. `cargo fmt && cargo clippy -- -D warnings && cargo test`
4. Open PR

## Release

Tag `v*` triggers automated publish to crates.io.
